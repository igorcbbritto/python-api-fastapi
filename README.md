# Python API FastAPI

API REST desenvolvida em Python utilizando FastAPI para consulta de status de servidores.

## Funcionalidades

- Endpoint de status do servidor
- Consulta de CPU, memória RAM e disco
- Health check da aplicação
- Documentação automática com Swagger

## Tecnologias

- Python
- FastAPI
- Uvicorn
- psutil
